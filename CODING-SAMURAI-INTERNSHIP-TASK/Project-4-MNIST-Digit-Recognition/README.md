# 🔢 Handwritten Digit Recognition (MNIST)

## 📌 Objective
Recognize handwritten digits using the MNIST dataset and a neural network.

## 📊 Dataset
MNIST dataset (60,000 training images, 10,000 test images)

## 🛠️ Skills Used
- Neural Networks (Keras/TensorFlow)
- Image Normalization
- Accuracy and Loss Visualization

## 📁 Files
- `digit_recognition.ipynb`: Neural network training code
- `requirements.txt`: Libraries needed

## ▶️ How to Run
1. Install dependencies using pip.
2. Run the notebook to train and evaluate the model.
