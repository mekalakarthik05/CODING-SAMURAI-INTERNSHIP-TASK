# 🔁 Customer Churn Prediction

## 📌 Objective
Predict if a customer will churn based on demographic and behavioral data.

## 📊 Dataset
Churn dataset (e.g., Telco Customer Churn from Kaggle)

## 🛠️ Skills Used
- Feature Engineering
- Classification (Random Forest, Logistic Regression)
- Model Evaluation (Confusion Matrix, ROC-AUC)

## 📁 Files
- `churn_prediction.ipynb`: ML model for churn prediction
- `churn_data.csv`: Dataset used

## ▶️ How to Run
1. Load the notebook.
2. Execute the code to preprocess data and train the model.
