# 🖼️ Image Classification Using Transfer Learning

## 📌 Objective
Classify images using a pre-trained model like ResNet or VGG16.

## 📊 Dataset
Custom image dataset or standard dataset like CIFAR-10 or Flower Classification

## 🛠️ Skills Used
- Transfer Learning with Keras
- Image Augmentation
- Deep Learning and Evaluation

## 📁 Files
- `transfer_learning.ipynb`: Code using pre-trained model
- `images/`: Folder containing training/testing images

## ▶️ How to Run
1. Install TensorFlow/Keras.
2. Run the notebook to train and evaluate the classifier.
